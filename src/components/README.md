# Design System Components

This directory contains reusable design system components built with React Aria Components and Tailwind CSS.

## Structure

- `ui/` - Basic UI components (Button, Link, Input, etc.)
- `layout/` - Layout components (Container, Grid, etc.)
- `navigation/` - Navigation components
- `feedback/` - Feedback components (Modal, Toast, etc.)

## Guidelines

- Use React Aria Components for accessibility
- Style with Tailwind CSS utility classes
- Use `clsx` for conditional className management
- Export components as named exports
- Include TypeScript types
- Follow mobile-first design principles
